## Реалізовані типи

Complex - містить реальну й уявну частини.

Vector3D - представляє вектор у тривимірному просторі.

Matrix2 - матриця розміром 2х2.

## Реалізовані функції (Complex)

add - додає два комплексних числа.

reduce - віднімає два комплексних числа.

multiply - множить два комплексних числа.

conjugate - повертає спряжене комплексне число до заданого.

abs_complex - модуль комплексного числа.

## Реалізовані функції (Vector3D)

norm - повертає норму вектору.

scalar_m - обчислює скалярний добуток двох векторів.

vector_m - повертає вектор, отриманий шляхом векторного добутку двох векторів.

mix_m - обчислює змішаний добуток трьох векторів.

## Реалізовані функції (Matrix2)

det - повертає визначник матриці.

inverse - повертає обернену до заданої матриці.

matrix_multiply - добуток двох матриць.
